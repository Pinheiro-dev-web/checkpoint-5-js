# Checkpoint V - Calculadora Interativa em JavaScript

Este repositório contém a resolução do **Checkpoint V** para a disciplina de Desenvolvimento Web (UNINOVE 2026.1), focado na manipulação do DOM e tratamento de eventos com JavaScript baseados na Aula 06.

## Funcionalidades e Implementações:
- **Operações Básicas:** Realiza soma, subtração, multiplicação e divisão utilizando a estrutura condicional `switch/case`.
- **Captura de Eventos:** Implementado através do método `addEventListener("click", ...)` acoplado ao botão de execução.
- **Validação de Erros:** Verificação de campos numéricos vazios usando `isNaN()` e bloqueio de divisão por zero.
- **Manipulação Dinâmica do DOM:** Atualização do conteúdo do card em tempo real alterando as propriedades de estilo e a propriedade `.textContent` dos elementos HTML de resposta.
